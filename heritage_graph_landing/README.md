# Heritage Graph Landing Page

A stunning, animated Next.js landing page that celebrates the intersection of cultural heritage and knowledge graphs. This project showcases how ancient wisdom can be connected through modern technology, featuring a beautiful Himalayan-inspired design with interactive elements.

